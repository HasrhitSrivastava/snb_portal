# SNB Portal

The **SNB Portal** is a web-based school management application built with Ruby on Rails. It helps schools manage academic and administrative activities such as student records, teacher data, classes, and attendance through a centralized platform with SNB admin.

---

## Ruby Version

- Ruby `3.2.3`
- Rails `8.1.1`

---

## System Dependencies

Make sure the following are installed on your system:

- Ruby
- Rails
- Bundler
- Node.js
- Yarn
- PostgreSQL
- Git

---

## Configuration

Clone the repository and install dependencies:

```bash
git clone https://github.com/HarshitSrivastava/snb_portal.git
cd snb_portal
bundle install
yarn install
```

## Database Creation

Create the database for the application using the following command:

```bash
rails db:create
```

## Database Initialization

```bash
rails db:migrate
rails db:seed
```
